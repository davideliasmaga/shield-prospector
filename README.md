# SHIELD Prospector

Extensao Chrome para geracao de mensagens de prospeccao no LinkedIn e Sales Navigator.

## Setup (uma vez so)

1. Va em Settings > Pages > Source: "Deploy from branch" > branch: main > folder: / (root)
2. Aguarde ~2 minutos para o GitHub Pages ativar

## Estrutura

- `extension/` — codigo da extensao
- `updates.xml` — arquivo de auto-update que o Chrome consulta
- `extension.crx` — pacote compilado da extensao (gerado automaticamente)
